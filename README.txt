Crossword Express is a simple program to create html code fast when we have crosswords on a table.
Firstly, you put the number of rows and columns, and then it creates a table...
Secondly you write the correct letters in the cells, that they will be the answers and then click on generate Table button
After this click generate results.

Well Bingo! You have the html code for the table, and the results (correct answers per answer box) in the right order
